# 🎸 baseloop — 베이스 루프 자동 생성기

장르 스타일 또는 **코드 진행**으로 베이스 라인을 만들고, 브라우저에서 **즉시 재생**하고,
악보·WAV·MP3·MIDI·LilyPond로 내보내는 웹 앱입니다.

데모: https://baseloop.onrender.com/

---

## ✨ 주요 기능

### 🎵 음악성 강화
- **코드 진행 따라가기**: `C G Am F`, `Dm7 G7 Cmaj7` 같은 진행을 입력하면 각 코드(=1마디)를
  근음·3음·5음·7음과 다음 코드로 향하는 **접근음(approach note)** 으로 연결해 "진짜 베이스 라인"에 가까운 결과를 만듭니다.
- **슬래시(온음) 코드**: `C/E`, `D/F#` 처럼 베이스 음이 지정된 코드는 마디 첫 박을 그 베이스 음으로 연주하고,
  접근음도 다음 코드의 실제 베이스 음을 향합니다. (베이스 라인에 가장 중요한 부분)
- **장르별 그루브 템플릿**: 락·펑크·팝·재즈(워킹 베이스)·블루스·레게·힙합 각각의 리듬 패턴.
- **스윙**: 0–80% 슬라이더로 오프비트 8분음표를 밀어 셔플 필을 적용.
- **시드(seed)**: 동일 입력 + 동일 시드 → 항상 동일한 결과(재현·공유 가능). 비워두면 서버가 랜덤 시드를 만들어 돌려줍니다.

### 🔊 클라이언트 오디오 (Tone.js)
- 서버 WAV 생성에 의존하지 않고 **브라우저에서 Tone.js로 즉시 합성·재생**합니다. (재생 지연/캐싱 이슈 없음)
- **리얼 베이스**: GM 사운드폰트(`electric_bass_finger`)의 실제 녹음 샘플을 `Tone.Sampler`로 로드해
  진짜 베이스 음색으로 재생합니다. 샘플 로드가 실패/지연되면 자동으로 신스 사운드로 폴백하므로 재생이 끊기지 않습니다.
  토글을 끄면 **초창기 합성 음색**(배음 1·0.4·0.2·0.1 + 부드러운 ADSR + 저역 부스트)으로 재생합니다.
- 재생 커서가 `Tone.Transport` 타임라인을 따라 악보 위에서 실시간으로 움직입니다.
- 마스터 체인(게인 → 리미터)으로 베이스+드럼+메트로놈 합산 시 클리핑을 방지합니다.

### 💾 저장 · 공유 · 편의
- **WAV / MP3 다운로드**: `Tone.Offline`로 클라이언트에서 렌더링 후 자체 WAV 인코더 / lamejs(MP3)로 저장.
- **MIDI / LilyPond 내보내기**: 서버에서 생성.
- **메트로놈 · 드럼 반주** 토글: 장르별 킥/스네어/하이햇 패턴으로 함께 재생.
- **URL 공유**: 현재 설정 전체를 URL 해시(`#s=...`)에 담아 "링크 복사" 한 번으로 공유. 링크를 열면 그대로 복원됩니다.
- **히스토리**: 생성한 베이스 라인을 브라우저(localStorage)에 자동 저장하고 클릭으로 즉시 불러오기.

### 🎼 악보
- VexFlow로 전문 표기, 실패 시 Canvas 폴백.
- Music21 + MuseScore가 설치된 환경에서는 "전문 악보 보기"로 고품질 SVG 악보를 생성(없으면 텍스트 악보로 폴백).

## 🚀 실행

```bash
pip install -r requirements.txt
python app.py            # 기본 포트 10000 (PORT 환경변수로 변경)
# 또는
gunicorn app:app
```

브라우저에서 `http://localhost:10000` 접속.

### 환경 변수
| 변수 | 설명 |
|------|------|
| `PORT` | 서버 포트 (기본 10000) |
| `FLASK_DEBUG` | `true`면 디버그 모드 |
| `MUSESCORE_PATH` | MuseScore 실행 파일 경로(전문 악보 SVG용). 없으면 자동 탐색 |

> Tone.js·VexFlow·lamejs는 CDN으로 로드되므로 별도 설치가 필요 없습니다.
> MuseScore가 없어도 나머지 기능은 모두 동작합니다(전문 악보는 텍스트로 폴백).

## 🧪 테스트

```bash
python -m unittest discover -s tests
# 또는
pytest
```

`tests/test_app.py`는 음표 주파수 계산, 시퀀스 파싱, 코드 파싱, 그루브 템플릿(모든 템플릿이 4박을 채우는지),
시드 재현성, WAV 버퍼, Flask 라우트(`/generate_notes` 랜덤·코드 모드, `/generate_audio`, `/generate_midi` 등)를 검증합니다.


## 🎹 시퀀스 형식

`음표이름+옥타브 박자단위`를 쉼표로 구분합니다. 예: `C2 1.0, G2 0.5, Eb2 0.5, F2 1.0` (`#`·`b`, 쉼표 `R 1.0` 지원).
옥타브 규약은 C5 = MIDI 60 입니다.
